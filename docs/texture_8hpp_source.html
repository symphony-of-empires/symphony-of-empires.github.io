22 // Abstract:
23 // This file implements a texture which is based from the binary image
24 // class to avoid code repetition. The difference of the texture from the
25 // binary image is that the texture is oriented towards OpenGL rendering
26 // more than the aforementioned binary image. A binary image is usable on
27 // any context but rendering, while the texture is intended to be used in
28 // rendering cases only.
